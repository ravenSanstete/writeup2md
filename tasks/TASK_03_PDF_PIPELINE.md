# TASK_03: PDF Native Extraction

## Objective

Implement PDF ingestion and native-content extraction into the same IR used by URLs.

## Required implementation

- source preservation and hashing;
- native text extraction;
- reading-order candidates;
- page and bounding-box provenance;
- embedded-image extraction;
- configurable page rendering;
- visual-block creation;
- pure-text Markdown from native blocks;
- scanned-page detection and unresolved status.

Do not implement PaddleOCR-VL enrichment in this task.
